## üí° Tips for Creating Effective Prompts

1. **Be Specific**: Include details about setting, lighting, camera angles, and mood
2. **Use Descriptive Language**: Rich adjectives help convey the exact visual you want

6. **Set the Atmosphere**: Include weather, time of day, lighting conditions
