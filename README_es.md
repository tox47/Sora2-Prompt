## 🎬 Introducción

Bienvenido a la **Galería de Prompts de Sora 2** - una colección completa de 100 prompts seleccionados de texto a video para el modelo de IA Sora 2 de OpenAI. Este repositorio muestra las increíbles capacidades de Sora 2 para generar videos de alta calidad a partir de descripciones de texto.

### ¿Qué es Sora 2?

Sora 2 es el modelo avanzado de IA de texto a video de OpenAI que puede crear escenas realistas e imaginativas a partir de instrucciones de texto. Comprende física compleja, animaciones detalladas de personajes y puede producir contenido de video de calidad cinematográfica.

## 📊 Colección de Prompts

Esta colección incluye:
- ✨ **20 prompts oficiales** de [SoraInviteCode.org](https://sorainvitecode.org/prompt)
- 🎨 **80 prompts adicionales** de varias fuentes comunitarias y creadores
- 🎯 Categorías: Animación, Escenas Realistas, Fantasía, Acción, Efectos Cameo y más

## 💡 Consejos para Crear Prompts Efectivos

1. **Sea Específico**: Incluya detalles sobre el escenario, iluminación, ángulos de cámara y ambiente
2. **Use Lenguaje Descriptivo**: Adjetivos ricos ayudan a transmitir la visión exacta que desea
3. **Especifique el Estilo**: Mencione estilos artísticos (anime, cinematográfico, fotorrealista, etc.)
4. **Incluya Detalles Técnicos**: Tipo de cámara, película, composición de plano
5. **Describa el Movimiento**: Especifique cómo deben moverse los objetos y personajes
6. **Establezca la Atmósfera**: Incluya clima, hora del día, condiciones de iluminación
7. **Agregue Contexto**: Elementos de fondo, comportamiento de multitudes, detalles ambientales
